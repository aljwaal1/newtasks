# Smart Tasks — Android

تطبيق مهام عربي محلي يعمل من Android 5.0 فأعلى، مع تنبيهات دقيقة وشاشة منبّه كاملة وصوت واهتزاز وتأجيل وسجل تشخيص.

## الإصدار الحالي: 1.2.0

### إدارة المهام

- إضافة وتعديل وحذف وإنجاز وإعادة فتح المهام.
- زر إنجاز كبير وواضح داخل بطاقة المهمة.
- عنوان وملاحظات وتصنيف وأولوية وتاريخ ووقت وتكرار.
- حالة المهمة مبسطة إلى: قيد التنفيذ أو مكتملة.
- الأولويات الافتراضية: عادية، متوسطة، عاجلة.
- إمكانية إضافة أولويات أخرى من الإعدادات وحذف غير المستخدمة منها.
- تحويل الأولويات القديمة تلقائيًا إلى النظام الجديد.
- تكرار يومي وأسبوعي وشهري وسنوي.
- بحث وتصفية حسب الحالة والأولوية.
- لوحة رئيسية وقائمة مهام وتقويم وإعدادات.
- تصنيفات قابلة للإضافة والحذف.
- تخزين محلي ونسخ احتياطي واستيراد وتصدير JSON.

### التاريخ والوقت

- منتقي تاريخ مرئي.
- إدخال الوقت في خانتين منفصلتين: الساعة والدقائق.
- الساعة تقبل من 00 إلى 23، والدقائق من 00 إلى 59.
- الانتقال تلقائيًا إلى خانة الدقائق بعد إدخال الساعة.
- رفض القيم غير الصحيحة ومنع حفظ تنبيه في الماضي.
- اختصارات لليوم وغدًا وبعد أسبوع.
- عرض الأرقام بصيغة 0–9 بدل الأرقام الهندية.

### التنبيهات

- `AlarmManager` لمواعيد المهام الدقيقة.
- إشعار من فئة المنبّه مع `FullScreenIntent`.
- تشغيل شاشة المنبّه عبر `PendingIntent` متوافق مع قيود التشغيل من الخلفية في Android الحديث.
- صوت منبّه غير متزامن لتجنب تجميد الواجهة.
- اهتزاز، تم الإنجاز، تأجيل 5 دقائق، وتأجيل 10 دقائق.
- إعادة تثبيت المواعيد بعد إعادة تشغيل الهاتف وتغيير الوقت والمنطقة الزمنية.
- مسارات احتياطية عند منع الخدمة أو الشاشة الكاملة.

### التشخيص

يسجل التطبيق المراحل الأساسية، ومنها:

```text
ALARM_SCHEDULE_REQUEST
ALARM_REGISTERED
ALARM_RECEIVED
ALARM_SERVICE_STARTED
NOTIFICATION_POSTED
ALARM_ACTIVITY_PENDING_INTENT_SENT
ALARM_ACTIVITY_OPENED
SOUND_PREPARING
SOUND_STARTED
VIBRATION_STARTED
```
