# Smart Tasks — Android

تطبيق مهام عربي محلي يعمل من Android 5.0 فأعلى، مع واجهة موحدة وتنبيهات دقيقة قابلة للتحكم.

## الإصدار الحالي: 1.3.0

### واجهة موحدة

- شاشة رئيسية واحدة دون شريط تنقل بين عدة شاشات.
- الإحصاءات والبحث والفلاتر وقائمة المهام ظاهرة في الواجهة نفسها.
- لا يوجد تمرير أفقي في الواجهة النشطة.
- الشاشة الرئيسية ثابتة، ويقتصر التمرير الرأسي على قائمة المهام عندما يزيد عددها عن مساحة الشاشة.
- الإضافة والتعديل والإعدادات تظهر كنوافذ فوق الشاشة الرئيسية.
- نافذة المهمة مضغوطة وتعرض الحقول الأساسية دون تمرير رأسي أو أفقي.

### إدارة المهام

- إضافة وتعديل وحذف وإنجاز وإعادة فتح المهام.
- زر إنجاز كبير وواضح داخل بطاقة المهمة.
- عنوان وملاحظات وتصنيف وأولوية وتاريخ ووقت وتكرار.
- حالة المهمة: قيد التنفيذ أو مكتملة.
- الأولويات الافتراضية: عادية، متوسطة، عاجلة.
- إمكانية إضافة أولويات أخرى وحذف غير المستخدمة منها.
- بحث وتصفية حسب الحالة والتاريخ والأولوية.
- تخزين محلي ونسخ احتياطي واستيراد وتصدير JSON.

### التاريخ والوقت

- منتقي تاريخ مرئي.
- إدخال الوقت في خانتين منفصلتين.
- الدقائق تظهر في موضع الساعة السابق، والساعة في موضع الدقائق السابق.
- الساعة تقبل من 00 إلى 23، والدقائق من 00 إلى 59.
- العرض والحفظ النهائي يبقيان بصيغة الساعة:الدقائق.
- رفض القيم غير الصحيحة ومنع حفظ تنبيه في الماضي.
- عرض الأرقام بصيغة 0–9 بدل الأرقام الهندية.

### طرق التنبيه

- بدون صوت مع اهتزاز، ويتوقف تلقائيًا.
- صوت خفيف مرة واحدة مع اهتزاز، وهو الوضع الافتراضي.
- صوت منبه عادي مع اهتزاز بحد أقصى 30 ثانية.
- لا يستخدم التطبيق نغمة منبه الساعة الافتراضية.
- جلسة تنبيه واحدة تمنع تداخل الأصوات.
- زر إيقاف فوري داخل الإشعار وشاشة التنبيه.
- شاشة التنبيه لا تعيد تشغيل الخدمة عندما تكون عاملة.

### التوافق

- minSdk 21: Android 5.0 فأعلى.
- APK عالمي واحد.
- لا توجد مكتبات أصلية خاصة بمعمارية واحدة.
- إعدادات البناء تشمل armeabi-v7a وarm64-v8a وx86 وx86_64.

## البناء

- JDK 17
- compileSdk 36
- targetSdk 36
- GitHub Actions ينفذ Android Lint ثم يبني APK تجريبيًا ويرفع التقرير وملف APK.
